# 🎠 游乐园照片查找系统 - 启动说明

## 项目版本说明

我为您准备了两个版本的系统：

### 📱 简化版本（推荐）
**适合快速体验，无需安装任何依赖**

- **优点**：单文件HTML，直接在浏览器中打开即可使用
- **功能**：包含核心的照片查找、预览和购买功能
- **使用方法**：直接双击 `simple-version/index.html` 文件

### 🏗️ 完整版本（前后端分离）
**功能完整，但需要安装Node.js和依赖包**

- **优点**：功能完整，支持用户认证、购物车、支付等高级功能
- **缺点**：需要安装Node.js和大量依赖包，启动时间较长
- **使用方法**：运行 `./start.sh` 脚本

## 📱 简化版本使用方法

1. **打开文件浏览器**，找到项目目录
2. **进入 `simple-version` 文件夹**
3. **双击 `index.html` 文件**，系统会自动在默认浏览器中打开
4. **开始使用**！您可以：
   - 🔍 使用搜索功能查找照片
   - 📸 浏览和预览照片
   - 🛒 将照片添加到购物车
   - 🎯 查看热门项目照片

## 🏗️ 完整版本使用方法（可选）

### 1. 环境要求
- **Node.js 16.x 或更高版本**
- **npm 8.x 或更高版本**
- **至少 2GB 可用内存**

### 2. 安装依赖（耗时较长）
```bash
# 安装后端依赖
cd backend
npm install

# 安装前端依赖  
cd ../frontend
npm install
```

### 3. 启动服务
```bash
# 在项目根目录运行
./start.sh
```

### 4. 访问地址
- **前端应用**: http://localhost:5173
- **后端API**: http://localhost:3001

## 🎯 主要功能介绍

### 核心功能
- **🔍 照片查找**: 按日期、地点等条件搜索照片
- **📸 照片预览**: 查看低清预览版本（带水印）
- **🛒 购物车**: 管理要购买的照片
- **💰 购买功能**: 支持单张购买和套餐购买
- **🔥 热门项目**: 查看最受欢迎的拍照项目

### 高级功能（完整版本）
- **👤 用户认证**: 注册、登录、个人中心
- **🔐 安全支付**: 支持多种支付方式
- **🤖 AI智能匹配**: 上传自拍照进行人脸识别匹配
- **📊 订单管理**: 查看和管理历史订单
- **📱 响应式设计**: 完美适配手机、平板和电脑

## 💡 提示和建议

1. **如果您只是想快速体验**，强烈建议使用简化版本
2. **如果您需要完整功能**，请确保网络稳定并预留足够的安装时间
3. **简化版本已经包含了所有核心功能**，对于大多数用户来说已经足够使用
4. **完整版本适合开发人员**或需要二次开发的用户

## ❓ 常见问题

**Q: 简化版本和完整版本有什么区别？**
A: 简化版本是单文件HTML，无需安装即可使用；完整版本是前后端分离架构，功能更完整但需要安装依赖。

**Q: 为什么完整版本安装依赖这么慢？**
A: 完整版本使用了React、Vite等现代前端工具，需要安装大量依赖包，建议耐心等待。

**Q: 简化版本的数据是真实的吗？**
A: 简化版本使用的是模拟数据，用于演示系统功能。

**Q: 如何切换到完整版本？**
A: 安装好依赖后，运行启动脚本即可切换到完整版本。
